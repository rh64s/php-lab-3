## Настройки и дополнительное

В `deploy` лежит база данных, которая использовалась.

В `config/path.php` -> 'root' можно указать строку, которая будет работать как префикс ко всем запросам (если сервер прослушивает не сам проект, а папку, например)

В `config/db.php` настройка базы данных (работа шла с `mariadb`/`mysql`)
